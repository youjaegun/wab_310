선택자의종류

html 파일 css 를 통해 꾸밀수 있고 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자{ 속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * html 내부의 모든태그를 불러서 결정한다.
태그선택자 태그명 html 내부의불려진 모든 태그를 설정
아이디선택자 #아이디이름 속성값 ID의 이름을 불러서 특정ID 만 설정
클래스선택자 .클래스명 속성값 class 의 이름을 불러서 특정클래스만 설정
id와 class의 차이점:
id는 한페이지 한번만 사용 중복사용안됨 한번사용한 id는 두번사용안됨
class 는 중복적용가능 , 한클래스에 여러 클래스명 사용가능 클래스명 한칸띄어쓰기로 여러 클래스명부여가능
중복선택자 , 여러개의 선택자를 나열해서 사용할수 있다.
중복선택자 붙여쓰기(태그.클래스명) 지정된태그중 해당 클래스이름만부름
후손선택자 선택자 선택자 부모자 내부에있는 요소를 모두 부른다.
자손선택자 부모자>자손선택자 다이렉트 자손자에 한해서 설정가능
